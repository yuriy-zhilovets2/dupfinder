Сравнение изображений с помощью CNN (convolutional neural network)

Хэш представляет собой вектор из 576 чисел float32
Сравнение происходит путем расчета косинуса угла между векторами (чем ближе к 1, тем изображения похожей)

Для расчета хэшей используется библиотека imagedeup https://github.com/idealo/imagededup

Файлы:
test-cnn-samples2.py Расчет сходства между файлами в каталоге samples/set2 (каждого с каждым)
test-faiss.py - То же самое, но с хранением полученных хэшей с помощью индексов FAISS
